Description

An I/O-independent MQTT protocol state machine.

  • Manages connection status.

  • Manages packet identifiers.

  • Manages Topic Aliases.

  • Manages packet resending on reconnection.

  • Manages PINGREQ/PINGRESP timers.

  • Manages automatic response packet sending.

All requests and settings are implemented as synchronous functions. When caller action is required, a std::vector of basic_event_variant corresponding to the required actions is returned.
