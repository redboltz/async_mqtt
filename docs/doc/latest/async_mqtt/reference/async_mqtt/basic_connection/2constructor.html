async_mqtt::basic_connection::basic_connection

Constructor.

Synopsis

Declared in header <async_mqtt/protocol/connection.hpp>

basic_connection(
    protocol_version ver,
    basic_packet_id_type<PacketIdBytes>::type packet_id_max = std::numeric_limits<typename basic_packet_id_type<PacketIdBytes>::type>::max());

Description

  • If the connection is a client, you must set either protocol_version::v3_1_1 or protocol_version::v5 .

    • If protocol_version::v3_1_1 or protocol_version::v5 is specified, only packets compatible with that protocol version can be sent and received.

  • If the connection is a server, you can also set protocol_version::undetermined . The protocol version is determined when the CONNECT packet is received. Once the protocol version is determined, only packets corresponding to that protocol version can be sent and received.

Parameters

Name Description

ver

The protocol version.

packet_id_max

The maximum packet_id that can be used. It is only for test purpose. By default, it is set to the maximum value of PacketIdBytes type. If PacketIdBytes is 2, the maximum packet_id is 65535. (MQTT protocol) If PacketIdBytes is 4, the maximum packet_id is 4294967295.
