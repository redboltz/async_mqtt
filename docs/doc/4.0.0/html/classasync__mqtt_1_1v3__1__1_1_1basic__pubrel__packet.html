Detailed Description

template<std::size_t PacketIdBytes>
class async_mqtt::v3_1_1::basic_pubrel_packet< PacketIdBytes >

MQTT PUBREL packet (v3.1.1)

Template Parameters
PacketIdBytessize of packet_id

If basic_endpoint::set_auto_pub_response() is called with true, then this packet is automatically sent when PUBREC v3_1_1::basic_pubrec_packet is received. If both the client and the broker keeping the session, this packet is stored in the endpoint for resending if disconnect/reconnect happens. If the session doesn' exist or lost, then the stored packets are erased.
See http://docs.oasis-open.org/mqtt/mqtt/v3.1.1/os/mqtt-v3.1.1-os.html#_Toc398718053
